Import draft

To import a playersfile, you need to export the draft from Front Office Football using the Extractor utility in the FOF 2007 Utility Suite. This will be in a .csv file ("FOFRoster.csv" most likely).

NOTE: Be sure that the list is sorted by adjusted grade before exporting from FOF. This is the default sort order, but if you change the order in FOF then the players will be listed in that order. The adjusted grade is not exported in this file, so this utility will assume that the order that you exported from FOF is adjusted grade order.

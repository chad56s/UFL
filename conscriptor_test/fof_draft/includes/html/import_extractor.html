Import Extractor File

To import a playersfile, you need to export the draft from Front Office Football using the Extractor utility in the FOF 2007 Utility Suite. This will be in a .csv file ("FOFRoster.csv" most likely). When you import this file, the data from your interviews will be visible in all your player lists.

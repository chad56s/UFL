Import Mock Draft Data

To create a mock draft and allow for scout picks, you must first run Extractor on the entire league. To do that, open up a roster but do not open a player file. Run extractor and it will cycle through all the teams, generating an FOFRoster.csv file. This is the file you need to upload to set up the mock draft.

Once the file is uploaded, you will be given a page to link the team names to their respective 3-letter code, then the mock draft will run.

NOTE: If you did not upload the draft_preview.html file, you cannot run a mock draft and have scout picks.
